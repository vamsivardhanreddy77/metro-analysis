#Delhi Metro Data Analytics Dashboard

A Python data analytics project to explore and visualize insights from **Delhi Metro dataset**. This interactive analysis features data overview, visualizations, and machine learning-based predictions to understand crowd patterns and metro usage trends.

---

# Project Overview

This project analyzes the Delhi Metro dataset using Python and provides:

- Data cleaning and preprocessing
- Descriptive and visual analytics
- Machine learning-based crowd prediction
- Interactive visualizations
- Multiple data exploration modules

The goal is to extract meaningful insights into metro ridership behavior and provide analytical support for transportation studies.

---

## Features

### Data Overview
- Load and inspect dataset structure
- Handle missing and inconsistent data
- Display basic statistics and first look

### Visualizations
- Line charts, bar charts, scatter plots
- Ridership trends over time
- Station distribution and comparisons

### Crowd Prediction
- Predict crowd levels using a trained ML model
- Input features selected dynamically
- Outputs help estimate passenger footfall patterns

### Utility Modules
- Structured scripts for data loading
- Organized visualization functions
- Modular design for scalability
